✨ 主要功能
🌱
成長系統
悉心照顧的話，就會從嬰兒成長為大人喔
🤖
AI 台詞生成
Gemini AI 會親自創作出符合角色性格的台詞
💕
好感度系統
一起培育 2 隻角色，彼此之間就會產生關係
📔
AI 日記
每天 AI 都會以角色的視角撰寫一日日記
🏆
20 個成就
達成各種條件並收集成就吧
🌸
季節 & 時間
背景會根據實際的時間和季節進行變化
📖 使用方法
1
點擊開始按鈕來開啟彈出視窗。
如果出現封鎖快顯視窗，請在網址列中選擇允許。
2
點擊舞台的🐾 上傳圖片來放入角色圖片。
推薦使用 PNG 透明背景圖片！
3
在⚙ 設定中輸入名字、個性與 Gemini API 金鑰。
擁有 API 金鑰的話，AI 就會直接產生對話喔。
4
透過餵食、遊玩、撫摸來照顧角色吧！
如果放任不管的話數值會下降，記得要經常來看看牠們喔 🐾
🔑 Gemini API 金鑰獲取方法
1 前往 aistudio.google.com
2 使用 Google 帳戶登入
3 點擊左側選單的 Get API Key → Create API Key
4 複製生成的金鑰，並貼到 設定 → 共通 → API 金鑰！
💡 Gemini API 可以免費使用。
桌面寵物採用預先收集台詞的方式，因此 API 呼叫次數非常少。
即使沒有 API 金鑰，也能使用基本台詞來遊玩喔！
